Benson Programming Masterclass is an intensive Programming class which takes place around the various regions of Ghana periodically. The class is purely practical with the aim of creating programmers with a sharp practical mind. There two classes; The Web Development Class and the Programming Class

The Web Development class focuses on the basics of web development; HTML,CSS, and JavaScript. From beginner stage, students will be taught the basics of three mentioned technologies. Through itermedaite to Advanced stage, students will be taken through various CSS and JavaScript frameworks which include Bootstrap and React.js

Backend technologies are also taught but only the basics. That is, the basics of Node.js, SQL, MongoDB.

Programming Class

Here, the basics of progamming is taught from beginner to advanced. Students are taught whot computers are, how they communicate and more. Students are taught how to program in C++. Projects will be built and deployed.
